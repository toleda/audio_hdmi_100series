audio_hdmi_100series
============
OS X Skylake/100 Series/Socket 1151 - HD5x0/AMD/Nvidia HDMI Audio

Version
v1.0 - 3/31/2016, Initial HD5x0 support

HDMI audio for OS X Skylake systems with the Skylake HDMI audio ssdt.

HD5x0 Support
1. HD530/Desktop/AMD/Nvidia HDMI Audio (Audio ID: 3 = patch-A3 = filename-3, etc.)
2. HD540/NUC HDMI Audio (Audio ID: 1 = patch-A1 = filename-1, etc.)
